* bcrypt-nodejs
* body-parser
* connect-multiparty

* mongoose
* mongoose-pagination

Copy and paste in the terminal.
`npm install bcrypt-nodejs body-parser connect-multiparty express jwt-simple moment mongoose mongoose-pagination --save`
